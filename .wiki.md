# Project Summary
The project is an AI Short Drama Script Generator web application that allows users to create scripts in various genres using AI technology. It features a user-friendly interface, offering a script generation tool with a free usage limit and upgrade options. The application is designed for content creators looking to generate production-ready scripts efficiently.

# Project Module Description
- **Frontend**: Contains the user interface components, pages, and configurations for the web application.
- **Backend**: Manages the server-side logic, database interactions, and API endpoints.
- **Dependencies**: Includes necessary libraries and tools for both frontend and backend functionality.
- **Services**: Handles business logic and integrations, such as payment processing and data management.

# File Description Inventory
- `app/backend/main.py`: Main entry file for the backend application.
- `app/frontend/src/App.tsx`: Main application component for the frontend.
- `app/frontend/src/pages/`: Contains different pages of the web application.
- `app/backend/requirements.txt`: Lists Python dependencies for the backend.
- `app/frontend/package.json`: Lists JavaScript dependencies for the frontend.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Python, FastAPI, SQLAlchemy
- **Database**: PostgreSQL
- **Deployment**: Docker, Stripe for payment processing

# Usage
1. **Install Dependencies**:
   - For backend: Run `pip install -r requirements.txt`
   - For frontend: Run `pnpm install`
   
2. **Build the Application**:
   - Navigate to the frontend directory and run `pnpm run build`.

3. **Run the Application**:
   - Start the backend server using the command appropriate for your setup.
   - Serve the frontend files using a suitable web server.
